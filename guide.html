Справочник по Flexbox

Контейнер

display flex Устанавливает элемент как flex-контейнер
display inline-flex Устанавливает элемент как flex-контейнер, который может быть инлайновым

flex-direction

flex-direction row Располагает элементы в строку
flex-direction row-reverse Располагает элементы в строку в обратном порядке
flex-direction column Располагает элементы в столбец
flex-direction column-reverse Располагает элементы в столбец в обратном порядке

justify-content

justify-content flex-start Выравнивает элементы по левому краю контейнера
justify-content flex-end Выравнивает элементы по правому краю контейнера
justify-content center Выравнивает элементы по центру контейнера
justify-content space-between Распределяет элементы с равным пространством между ними
justify-content space-around Распределяет элементы с равным пространством вокруг них
justify-content space-evenly Распределяет элементы с равным количеством пространства между ними, включая краевые элементы

text-align

text-align left Текст выравнивается по левому краю элемента (значение по умолчанию)
text-align center Текст выравнивается по центру элемента
text-align right Текст выравнивается по правому краю элемента
text-align justify Текст выравнивается по ширине элемента, так что каждая линия занимает всю ширину, за исключением последней, которая выравнивается по левому краю
